Better handling of large data sets

TopCount data

BioDare2 is a repository for circadian, biological data, providing a platform for data sharing and period analysis. The main features of BioDare2 are: fast period analysis, timeseries processing, attractive visualizations, simple data description and modern UI, all of it to assure the best user experience.

Watch our introductory video and subscribe to our channel for training materials.

You can contact us at: biodare (@) ed.ac.uk

You need a modern, up to date browser to use BioDare2 (we are testing it with Chrome).

BioDare2 is currently at beta version. Check the bottom of the page how this early stage may affect you.

Timeseries data

Circadian research is all about timeseries. BioDare can visualize your data and allows you to perform various timeseries processing:

  • four detrending methods
  • normalization
  • alignment


Available methods:

  • FFT NLLS
  • MESA
  • MFourFit
  • Enright Periodogram
  • Lomb-Scargle periodogram
  • Spectrum resampling

Rhythm analysis

Period analysis is the strongest point of BioDare. BioDare gives you access to the:

  • best analysis methods
  • attractive visualizations
  • result aggregation

BioDare2 is still under development. Please read below to find out how this may affect you.
 

Rhythm analysis

We are using exactly the same implementation of period analysis methods as in original BioDare.

These methods will not change so any results you obtain using this version will be the same as in the future ones.

Data

Your data is safe, changes to BioDare will not affect the deposited data.

At the moment data grouping and averaging is not available but we are working on it.

You

Unfortunately bugs and glitches are to be expected.

The error messages are vague and probably not very informative. Please have patience; refresh page if the system appears stuck; email bug reports to biodare (@) ed.ac.uk.

Lab groups, as in BioDare, will return with a further release.
